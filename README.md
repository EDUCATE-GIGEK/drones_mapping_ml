# Drone/Motor Computer Vision Technology
The goal of this project is to explore Computer Vision technology in robotics and see where it takes us


## Phase 1: Utilizing simulators
Here, we will use AdruPilot SITL, the DroneKit simulator, and waypoint scripts in Python to write code for the drone

**Project 1 1**
The drone should follow a list of GPS waypoints.

**Project 2**
Altitude hold based on terrain elevation

**Project 3**
Computer vision-guided landing on a target


## Phase 2:
We will synchronize the code from project 3 into an actual drone with a companion computer (e.g., a Raspberry Pi 4)

## Phase 3:
Develop a computer vision algorithm to allow for Autonomous gameplay in the Calvin robotic football club computers
